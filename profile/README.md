# Tech of Our Own

We build **non-extractive consumer technology**: local-first, user-owned, repairable, and auditable.

**Non‑negotiables:** no subscriptions, no surveillance business models, no hostage mechanics.

## Licensing posture (plain language)

  https://github.com/techofourown/org-techofourown/blob/main/docs/decisions/ADR-0001-adopt-agplv3-core-software.md

- **Core hardware designs**: CERN-OHL-S-2.0 (strong reciprocity for shipped modifications)
  https://github.com/techofourown/org-techofourown/blob/main/docs/decisions/ADR-0005-adopt-cern-ohl-s-2-0-core-hardware-designs.md
